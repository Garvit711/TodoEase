# 📝 TodoEase | Stop Juggling, Start Doing

**TodoEase** is a lightweight, high-performance task management application designed to take the mental load off your shoulders. Built with a "no-nonsense" philosophy, it provides a clean space to capture thoughts, manage goals, and maintain a disciplined daily routine without the clutter of over-complicated menus.

## 🚀 Key Features

* **⚡ Snappy Task Management:** Add and delete tasks in seconds with zero friction.
* **🔄 Automatic Prioritization:** Checked tasks automatically drop to the bottom, keeping your "To-Do" list fresh.
* **💾 Persistent Sessions:** The app retains your tasks even if you close the tab or refresh the page.
* **📱 Snappy Routing:** Built-in routing makes moving through the app feel natural and fast.

## ⚙️ Installation & Setup

To run this project locally, run the following commands in your terminal:

```bash
# 1.Clone the repository
git clone [https://github.com/Garvit711/TodoEase.git]

# 2.Navigate to the folder
cd TodoEase

# 3.Install dependencies
npm install

# 4.Start the app
npm start
